# Pluck Privacy Policy

This repository hosts the official privacy policy 
for the Pluck Chrome Extension.

## About Pluck

Pluck is a Chrome extension that extracts text 
from any screen area instantly using local OCR 
technology. Available on the Chrome Web Store.

## Privacy Summary

Pluck is built with privacy as the foundation.

- No personal data collected
- No servers involved
- No tracking or analytics
- No third party data sharing
- No account required
- Everything runs 100% locally in your browser
